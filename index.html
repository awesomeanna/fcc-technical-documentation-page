Introduction

Linux is a family of open-source Unix-like operating systems based on the linux kernel by Linus Torvalds. Popular Linux distributions include Debian, Fedora and Ubuntu. Linux is one of the most prominent examples of free and open-source software collaboration.

Basics and Navigation

Understanding how to use the command line is imperative. This helps in navigating around directories, copying files, creating files among others.

Home Directory

On Unix and Linux systems, every user has a "home" directory. Your home directory will be where your settings files get stored and where your desktop is kept. Finding the home directory is incredibly easy. Use the pwd command to get the path of your home directory.

$pwd

Present Working Directory

Understanding where you are at any given time in your file system is important. This can be done by using the pwd command. $pwd pwd will list the "logical" path of your current directory.

List Files and Directories

Listing helps you to see what files or directories are in your current directory, or another directory. To accomplish this, we use the ls command.

$ls

  • Listing a Different Directory. By default, ls will operate in the current directory that you're in. You may, however, be able to find out what files are in another directory without leaving the current directory. $ls /usr/local
  • Listing all Files. You can also view both "visible" and "invisible" or "hidden files" by using the -a flag. $ls -a
  • Long Form Listing - This lists out the names of the files and directories as well as give more detailed information about them. $ls -l
  • Human Readable Sizes - The -l flag aforementioned gives us a more detailed views of files and directories in our current directory, including the number of bytes in the file. It's, however, more useful to view the size of the file in human readable terms such as 1.7K or 35M. $ls -lh

Links

Links allow us to create an association from one file or directory to another. It allows us to "point" one file or directory to another. Links can either be "hard" or "symbolic" links.

  • Hard Links - Hard links create an identical copy of the linked file, that gets updated automatically as the source file gets updated. To create a link to a file, we call the ln command - first giving it the name of the file we want to link, followed by the name of the linked file we want to create. $ln a.txt b.txt
  • Symbolic Links - Unfortunately, hard links do not work for directories. We can, however, use the -s flag to create a symbolic link. This can be used to link files as well as directories. $ln -s a.txt b.txt

Change Directories

Eventually, you'll need to change or navigate into another directory. This is done using the cd command. cd takes the argument of the path to the directory you wish to navigate to. $cd ~/Downloads

You can navigate back to your home directory by calling the cd command without any arguments. $cd

Creating Directories

Directories are used to store and organize files. To create a directory, we can use the mkdir command followed by the name of the new directory. $mkdir Projects

The mkdir command supports a verbose mode flag, -v that is used to print the results of mkdir to the console. $mkdir -v Projects The following is the output: mkdir: created directory 'Projects'

Deleting Files

The rm command is used to delete files and folders. It supports verbose mode flag as well. $rm -v Projects
